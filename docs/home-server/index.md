# 🏠 Home Server

Narzędzia ułatwiające zarządzanie własnym serwerem domowym –
od paneli administracyjnych, przez reverse proxy, po monitoring i backupy.

{: .note }
Jeśli dopiero zaczynasz – zacznij od **Casaos** lub **Portainer** jako panelu zarządzania Dockerem,
oraz **Nginx Proxy Manager** jako reverse proxy z obsługą Let's Encrypt.

---

## Kategorie narzędzi

### Panele zarządzania
Graficzne interfejsy do zarządzania serwerem i kontenerami Docker.

| Narzędzie | Opis | Ocena |
|-----------|------|-------|
| Portainer | Zarządzanie Dockerem | ⭐⭐⭐⭐⭐ |
| CasaOS | Panel dla początkujących | ⭐⭐⭐⭐ |
| Homarr | Dashboard z widgetami | ⭐⭐⭐⭐ |

### Reverse Proxy
Przekierowanie ruchu HTTP/HTTPS do kontenerów z automatycznym SSL.

| Narzędzie | Opis | Ocena |
|-----------|------|-------|
| Nginx Proxy Manager | GUI dla nginx + Let's Encrypt | ⭐⭐⭐⭐⭐ |
| Traefik | Automatyczne wykrywanie kontenerów | ⭐⭐⭐⭐ |

### Monitoring
Kontrola zasobów serwera i alertów.

| Narzędzie | Opis | Ocena |
|-----------|------|-------|
| Uptime Kuma | Monitoring dostępności usług | ⭐⭐⭐⭐⭐ |
| Netdata | Monitoring zasobów w czasie rzeczywistym | ⭐⭐⭐⭐ |

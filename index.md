# 🖥️ Selfhosted PL

Katalog sprawdzonych narzędzi do samodzielnego hostowania.
Każde narzędzie zawiera opis, wymagania sprzętowe, link do dokumentacji i moją osobistą opinię.

{: .highlight }
Wszystkie narzędzia są open-source i bezpłatne. Hostujesz u siebie – Twoje dane zostają Twoje.
